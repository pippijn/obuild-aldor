\section{Algorithms part}
\defsec{algorithms}


The algorithms part of \LibCharSet contains the categories
\begin{itemize}
\item \adtype{DecompositionAlgorithmType},
\item \adtype{GroebnerBasisAlgorithmType},
\item \adtype{MedialSetAlgorithmType},
\item \adtype{ReducedGroebnerBasisAlgorithmType},
\item \adtype{SaturatedDecompositionAlgorithmType},
\item \adtype{SaturatedGroebnerBasisAlgorithmType},
\item \adtype{SaturatedReducedGroebnerBasisAlgorithmType},
\item \adtype{TriangularizationAlgorithmType},
\item \adtype{UpdatableMedialSetAlgorithmType}, and
\item \adtype{UpdatableTriangularizationAlgorithmType},
\end{itemize}

and the domains

\begin{itemize}
\item \adtype{AutoreducedSetTools},
\item \adtype{BasicSetSortedTools},
\item \adtype{BasicSetTools},
\item \adtype{ChiDecompositionTools},
\item \adtype{CoherentAutoreducedSetTools},
\item \adtype{CoherentComponentsDecompositionTools},
\item \adtype{GroebnerBasisTools},
\item \adtype{ReducedGroebnerBasisTools},
\item \adtype{SaturatedGroebnerBasisWrtPolynomialReductionTools}, and
\item \adtype{ZeroDimensionalChiDecompositionTools}.
\end{itemize}

These categories and domain can be grouped into triangularization and gröbner basis algorithms on the one hand and decomposition algorithms on the other. \reffig{algos} and \reffig{algos2} shows this separation.
