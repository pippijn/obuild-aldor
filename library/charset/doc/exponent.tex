\section{Terms}
\defsec{exponents}


\LibCharSet's terms part consists of the domains

\begin{itemize}
\item \adtype{ExponentCategoryTools},
\item \adtype{ClassPresortedDifferentialExponent},
\item \adtype{CumulatedExponent},
\item \adtype{ListExponent},
\item \adtype{ListSortedExponent}, and
\item \adtype{SortedListExponent}.
\end{itemize}

\adtype{ExponentCategoryTools} provides default implementations for several functions of \LibAlgebra's categories for terms (\adtype{ExponentCategory} and \adtype{GeneralExponentCategory}).

\adtype{ClassPresortedDifferentialExponent} stores terms via buckets of indeterminates of the same class.

\adtype{CumulatedExponent} stores terms as pairs of indeterminate and corresponding degree.

\adtype{ListExponent}, \adtype{ListSortedExponent}, and \adtype{SortedListExponent} store indeterminates via lists.
