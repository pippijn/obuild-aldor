\section{Computer algebra systems}

\adtype{ComputerAlgebraSystemTools},
\adtype{ComputerAlgebraSystemType},
\adtype{MapleTools},
\adtype{MathematicaFullFormTools}, and
\adtype{MathematicaFullFormParser} constitute the computer algebra system part of \projectname.

\adtype{ComputerAlgebraSystemType} is a category that encapsulates functions for converting between \Aldor's expressions and expressions in the syntax of a computer algebra systems. 

\adtype{MapleTools} implements \adtype{ComputerAlgebraSystemType} for the syntax of \Maple. 

\adtype{MathematicaFullFormTools} implements \adtype{ComputerAlgebraSystemType} for the syntax of \Mathematica. 

\adtype{MathematicaFullFormParser} is a \adtype{ParserReader} for the FullForm syntax of \Mathematica. This parser is used by \adtype{MathematicaFullFormTools}. 

\adtype{ComputerAlgebraSystemTools} is a repository for implementations of \adtype{ComputerAlgebraSystemType}s.
