\section{Exceptions}

\adtype{ArrayException},
\adtype{Exception},
\adtype{GeneratorException},
\adtype{FileException},
\adtype{ListException},
\adtype{NotFoundException},
\adtype{NotFoundExceptionType},
\adtype{SyntaxException},
\adtype{ReducibleModulusException},
\adtype{RuntimeError}, and
\adtype{TableException}
constitute the exceptions part of \projectname.

\adtype{ExceptionType} is a base category for exceptions.

\adtype{Exception} is a basic implementation of \adtype{ExceptionType}.

\adtype{NotFoundExceptionType} is a category for exceptions that indicate a failed search.

\adtype{NotFoundException} implements \adtype{NotFoundException}.

\adtype{RuntimeError} extends the builtin domain to \adtype{ExceptionType}.

The extensions of \adtype{ArrayException}, \adtype{GeneratorException}, \adtype{FileException}, \adtype{ListException}, \adtype{SyntaxException}, and \adtype{TableException} add \adtype{ExceptionType} to the corresponding domains of \LibAldor.

The extension \adtype{ReducibleModulusException} adds \adtype{ExceptionType} to the \adtype{ReducibleModulusException} domain of \LibAldor.
