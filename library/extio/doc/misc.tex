\section{Miscellaneous}

\adtype{ExtIOLibraryInformation},
\adtype{RuntimeStorage},
\adtype{StreamFormatter},
\adtype{StringTokenizer}, and
\adtype{Times}
constitute the miscellaneous part of \projectname.

\adtype{ExtIOLibraryInformation} provides information about \projectname in a machine (\Aldor) readable format.

\adtype{RuntimeStorage} is an interface to \Aldor's garbage collector.

\adtype{StreamFormatter} provides wrappers for streams allowing to format them. For example \file{linebreaker/linebreaker.as} shows how to break a stream into lines at a certain column.

\inputSourceFile{linebreaker/linebreaker.as}{An example of using \adtype{StreamFormatter}}
\inputOutput{linebreaker/linebreaker}

\adtype{StringTokenizer} allows to chop \adtype{String}{}s into a sequence of tokens.

\inputSourceFile{tokenizer/tokenizer.as}{An example of using \adtype{StringTokenizer}}
\inputOutput{tokenizer/tokenizer}

\adtype{Times} allows to determine how much time the system spends in a process, as illustrated in \file{times/times.as}.

\clearpage
\inputSourceFile{times/times.as}{An example of using \adtype{Times}}
\inputOutput{times/times}
